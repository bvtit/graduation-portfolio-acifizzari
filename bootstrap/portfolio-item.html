Project Description

The second half of sophomore year was the start of Cisco 1 after we finished getting PC Pro certified. To finish off Cisco 1, we were assigned a final project that had several parts. The parts were Letter of Intent, Invoice, Router Information, Switch Information, HWIC WAN Card Information, Subnetting Scheme, Configurations, Proof of Functionality, and Topology. The purpose of the project was to pretend we were the head of a networking company that would implement the network topology and subnet scheme we designed, buy the equipment needed, and set the configurations for each network device. The project gave us some real world experience of the process of buying the parts all the way to setting the configurations and the design. We also had to test the functionality of the network topology and see if everything was pinging properly, able to telnet into the network devices, and using tracert to test the connectivity of the whole network topology.

Project Details

  • Packet Tracer was used in desgining the topology and implementing configurations to the PCs, routers, and switches.
  • Researched information on the specific router and switch versions that were used in the topology, (Cisco 2901 Integrated Services Router and Cisco Catalyst 2960-24TT-L Switch)
